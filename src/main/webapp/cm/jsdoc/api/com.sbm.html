sbm

com. sbm

Author:
  • Inswave Systems

통신 함수(submission 또는 ajax 통신)를 작성한다.

Methods

(static) create(options)

Author:
  • Inswave Systems

Submission 객체를 동적으로 생성한다.

Example
com.sbm.create(options);
Parameters:
Name Type Description
options Object

Submission 생성 옵션 JSON 객체

Properties
Name Type Description
id String

submission 객체의 ID. 통신 모듈 실행 시 필요.
ref String

서버로 보낼(request) DataCollection의 조건 표현식.(조건에 때라 표현식이 복잡하다) 또는 Instance Data의 XPath.
target String

서버로 응답(response) 받은 데이터가 위치 할 DataCollection의 조건 표현식. 또는 Instance Data의 XPath.
action String

통신 할 서버 측 URI.(브라우저 보안 정책으로 crossDomain은 지원되지 않는다.)
method String

[default: get, post, urlencoded-post]

  • get : 파라메타를 url에 붙이는 방식 (HTML과 동일).
  • post : 파라메타를 body 구간에 담는 방식 (HTML과 동일)
  • urlencoded-post : urlencoded-post.
mediatype String

[default: application/xml, text/xml, application/json, application/x-www-form-urlencoded] application/x-www-form-urlencoded 웹 form 방식(HTML방식). application/json : json 방식. application/xml : XML 방식. text/xml : xml방식 (두 개 차이는 http://stackoverflow._com/questions/4832357 참조)
mode String

[default: synchronous, synchronous] 서버와의 통신 방식. asynchronous:비동기식. synchronous:동기식
encoding String

[default: utf-8, euc-kr, utf-16] 서버 측 encoding 타입 설정 (euc-kr/utf-16/utf-8)
replace String

[default: none, all, instance] action으로부터 받은 response data를 적용 구분 값.

  • all : 문서 전체를 서버로부터 온 응답데이터로 교체.
  • instance : 해당되는 데이터 구간.
  • none : 교체안함.
processMsg String

submission 통신 중 보여줄 메세지.
errorHandler String

submission오류 발생 시 실행 할 함수명.
customHandler String

submssion호출 시 실행 할 함수명.
submitHandler requestCallback

{script type="javascript" ev:event="xforms-submit"} 에 대응하는 함수.
submitDoneHandler requestCallback

{script type="javascript" ev:event="xforms-submit-done"} 에 대응하는 함수
submitErrorHandler requestCallback

{script type="javascript" ev:event="xforms-submit-error"} 에 대응하는 함수

(static) execute(sbmObj, requestData, compObj)

Author:
  • InswaveSystems

서버 통신 확장 모듈, Submission를 실행합니다.

Example
// Submission의 ref 속성에 서버에 전송할 DataCollection을 설정하는 경우
com.sbm.execute(sbm_search); 

//  Submission의 ref 속성에 서버에 전송할 DataCollection을 설정하지 않고 JSON 객체를 전송하는 경우
var searchParam = {
	id : "A0001"
};
com.sbm.execute(sbm_search, searchParam);
Parameters:
Name Type Description
sbmObj Object

submission 객체
requestData Object

[Default : null, JSON, XML] 요청 데이터로 submission에 등록된 ref를 무시하고 현재의 값이 할당된다.
compObj Object

[Default : null] 전송중 disable시킬 컴포넌트

	 target : 'data:json,{"id":"dlt_codeGrp","key":"data"}',
